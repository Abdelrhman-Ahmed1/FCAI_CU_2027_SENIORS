# FCAI-CU Class of 2027 Senior Profile Frame Generator 🎓

Celebrate your graduation! This is a simple, modern, client-side web application designed for FCAI-CU (Faculty of Computers and Artificial Intelligence, Cairo University) Class of 2027 Seniors. It allows students to upload their personal photos, crop/adjust them cleanly, and overlay the official graduation frame before downloading a high-quality profile picture.

---

## 🚀 Features

* **Image Upload:** Supports all standard image formats.
* **Precise Cropping:** Integrated with `Cropper.js` for an intuitive 1:1 (square) cropping experience—complete with zooming and dragging.
* **Seamless Overlay:** Blends the cropped user photo behind a transparent frame dynamically using HTML5 Canvas.
* **High-Resolution Download:** Exports a high-quality PNG (`1080x1080px`) ready for social media.
* **Completely Client-Side:** No server side or database required. Your photos remain completely private on your device.
